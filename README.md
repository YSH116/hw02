# hw02
首先將三個button以uLCD、電線和電阻連接於麵包板，並在main函式以uLCD的內建函式控制面板，輸出初始頻率。
另外算出低頻濾波器的電路的截止頻率和相對應的週期，並以DAC(D7)和ADC(D11)連結。
最後以picoscope在選完頻率時，畫出比原本三角形平滑的週期曲線，並和FFT的圖形做比較。
